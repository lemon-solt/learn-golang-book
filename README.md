# レスポンスをチェックする時

本書では、cURL を使用しているが、リクエスト情報を整理出来る事も考えると Postman の使用をおすすめします。

https://www.postman.com/

- ファイルを送信するなどの操作はダウンロード版で実施すると参照出来ないエラーが発生し、解決に手間が掛かりますので Web 版を利用する事をおすすめします。
